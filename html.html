HTML笔记

上次更新：2017年11月06日

表单

<form action="服务器文件" method="传送方式"(post/get)></form>

1.所有表单控件（文本框、文本域、按钮、单/复选框等）都必须放在 
<form></form>
标签之间（否则用户输入的信息提交不到服务器）。
2.method:post/get的区别属于后端考虑的问题。

文本域

<textarea rows="行数" cols="列数">文本</textarea>
row和col可以用CSS中的height和width代替。

复选框/单选框

<input type = 'radio' value = '值' name = '名称' checked = 'checked'>
1.type: radio : 单选框

同一组单选按钮，name取值一定要一致


2.value:提交数据到服务器的值

供后台php使用


3.name:为控件命名

供后台asp、php使用


4.checked：当设置checked = 'checked' 时，该选项被默认选中。

下拉列表框

<select><option value = '提交值'> 选项 </option></select>
1.提交值：向服务器提交的值；选项：显示的值。
2.slected = 'selected' ,该选项默认被选中。
3.复选 <select multiple = 'multiple'></select>

按钮提交

<input type = 'submit' value = '提交'>

按钮重置

<input type = 'reset' value = '重置'>
